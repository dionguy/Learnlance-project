Learnlance
==========

An interactive learning platform designed to empower students, educators, and lifelong learners.

Overview
--------

Learnlance is a modular, open-source educational toolkit that supports personalized learning experiences.
It offers interactive lessons, quizzes, progress tracking, and integration with popular learning management systems.

Installation
------------

To install Learnlance locally:

.. code-block:: bash

   git clone https://github.com/yourusername/learnlance.git
   cd learnlance
   pip install -r requirements.txt

Usage
-----

Start the development server:

.. code-block:: bash

   python manage.py runserver

Then visit `http://localhost:8000` in your browser.

Documentation
-------------

Full documentation is available at:

`Learnlance Docs <https://learnlance.readthedocs.io>`_

.. code-block:: bash

   cd docs
